Stereo Visual Odometry

Under the guidance of Prof.KS Venkatesh, Computer Vision Lab, IIT-Kanpur (Ongoing)

  • Implemented an inlier-detection based approach [Howard2008] and Model-based ICP appraoch [Jiang2014] in MATLAB.
  • Working on improving the accuracy of model-based ICP method, by introducing iterative optimization techniques like Levenberg-Marquardt.
  • Performance is being evaluated on the KITTI odometry benchmark.

Range Flow (RGBD version of Optical Flow)

Under the guidance of Prof.KS Venkatesh, Computer Vision Lab, IIT-Kanpur (Summer 2014)

  • Performed a literature review on Optical Flow, and the various approaches on extending it to 3D.
  • Captured RGBD Data from a Microsoft Kinect using OpenNI and OpenCV libraries, and programming in C++.
  • Implemented two approaches for estimation of Range Flow from RGB-D data, one using ’Local Least Squares Solution’, the other a ’Global Minimum Energy Solution’, and the above two approaches were evaluated for speed and accuracy.
  • A third approach, Physical RGBD Flow, attempting to capture the physical flow (instead of apparent flow) was also implemented, and was evaluated using ground truth data generated using a highly accurate 6-DOF tracking device, the Personal Space Tracker.

Landmark Based Localisation using a Kinect Sensor (Winter 2013)

  • Geometric Triangulation was used to determine the pose (modelled as a set of random variables with Gaussian Distribution) of a robot, from information obtained from noisy landmarks.
  • A Microsoft Kinect was used to identify the landmarks (using color histogram based models), CAMshift algorithm was used to track these landmarks, and bearing measurements were calculated using the depth data.
  • An error model for the data obtained from a Kinect was used along with the Error Propagation Law to arrive at the uncertainty in the final pose computed using the Geometric Localisation Algorithm.

Discrete Hilbert Transform in Verilog

  • Implemented a 32-point discrete Hilbert Transform in Verilog.
  • Used a sequential implementation of FFT and IFFT as intermediate steps in the calculation of the Hilbert Transform.
  • Results obtained were evaluated against the Hilbert transform obtained from Matlab.
  • Secured 2nd position in ECDC, Techkriti ’14(Annual Technical and Entrepreneurship Fes- tival of IIT-Kanpur).

Cashless Campus System

  • A POS device was prototyped on the Arduino platform.
  • A fingerprint sensor was used to authenticate the buyer.
  • A touch screen based interface guided the user thought the billing process.
  • The transaction was written on an on-board SD-card, and was also updated on a server using an Ethernet port interfaced to the Arduino microcontroller.
  • Invoice was printed using a thermal printer.

Snakes on a 2D LED Matrix

  • An 8x8 LED matrix was designed and fabricated, which was then controlled using an ATmega8 microcontroller following the principles of Persistance of Vision.
  • The classic mobile phone game “Snakes” was implemented with various levels.

Laser Tag

  • An infrared-based ‘gun’ was developed, which fired beams modulated at 38KHz.
  • The recieving jacket had TSOP sensors to detect the infrared beams fired at it.
  • In order to increase the range of the gun, an amplification cirucit employing MOSFETs was used, along with optical magnification via an convex lens.
